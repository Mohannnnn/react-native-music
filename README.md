### RN-Music
> 基于react-native开发的音乐🎵APP.

### Start
```shell
1.IOS `react-native run-ios`
2.Android `react-native run-android`
```

### 环境
* node  v10.13.0
* npm   v6.5.0
* yarn  v1.9.4
* Xcode v9.4
* JDK   v1.8.0
* SDK   v
